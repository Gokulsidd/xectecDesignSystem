# Design System Tokens

Welcome to the **Xectec Design System** token registry. This catalog is generated dynamically from the core token package. Developers can use these design tokens via:
1. **CSS Variables** (e.g., `var(--color-primary-600)`, `var(--spacing-button-padding-x-md)`)
2. **TypeScript / JavaScript** (e.g., `tokens.colors.primary[600]`, `tokens.components.button.spacing.paddingX.md`)

## 🧩 Component Spacing & Color Overrides

These tokens are mapped to specific component requirements, allowing developers to set uniform layouts and structures.

### Button Spacing
* **Small Button**: `padding: var(--spacing-button-padding-y-sm) var(--spacing-button-padding-x-sm)`
* **Medium Button (Default)**: `padding: var(--spacing-button-padding-y-md) var(--spacing-button-padding-x-md)`
* **Large Button**: `padding: var(--spacing-button-padding-y-lg) var(--spacing-button-padding-x-lg)`
* **Icon to Text Gap**: `gap: var(--spacing-button-gap)`

### Card Padding Options
* **Small Card**: `padding: var(--spacing-card-padding-sm)` (12px)
* **Medium Card**: `padding: var(--spacing-card-padding-md)` (20px)
* **Large Card**: `padding: var(--spacing-card-padding-lg)` (32px)

### Input Field Paddings
* **Small Field**: `padding: var(--spacing-input-padding-y-sm) var(--spacing-input-padding-x-sm)`
* **Medium Field**: `padding: var(--spacing-input-padding-y-md) var(--spacing-input-padding-x-md)`
* **Large Field**: `padding: var(--spacing-input-padding-y-lg) var(--spacing-input-padding-x-lg)`
